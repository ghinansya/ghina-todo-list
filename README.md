- Setup dan Install
Requirements :
  
    Node.js (npm, npx)

    Browser : Chrome, Firefox, Edge, etc

Clone Repository
```html
git clone https://github.com/ghinansya/ghina-todo-list.git
```
```ruby
cd/ghina-todo-list
```

Menginstal package dalam proyek
```ruby
npm install
```

Menjalankan proyek
```ruby
npm start
```

Proyek akan berjalan di local dengan url:
```ruby
https://127.0.0.1:3000
```
