### Mobile Programming 2 - ToDo-List App

- Setup dan Install

Clone Repository
```bash
https://github.com/ghinansya/ghina-todo-list.git
```
```bash
cd/ghina-todo-list
```

Menginstal package dalam proyek
```bash
npm install
```

Menjalankan proyek
```bash
npm start
```

Proyek akan berjalan di local dengan url:
```bash
https://127.0.0.1:3000
```
